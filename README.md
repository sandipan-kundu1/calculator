# Simple React Calculator

A minimal calculator web app built with **React** and **Vite** that supports basic arithmetic operations: addition, subtraction, multiplication, and division.

## Features
- Clean, responsive UI
- Handles decimal numbers
- Clear (AC) button to reset
- Error handling for division by zero

## How to Run Locally
```bash
# Install dependencies
npm install

# Start development server
npm dev   # or: npm run dev
```
Open http://localhost:5173 in your browser.

## Build for Production
```bash
npm run build   # outputs to ./dist
```
Preview the build:
```bash
npm run preview
```
